## HTML Boilerplate
Front-end boilerplate for faster and easier web development

### Components
* Initializr 4.0
* Twitter Bootstrap 2.3.1 (grid, layouts, visibility classes and media queries)
* Modernizr 2.6.2
* jQuery 1.9.1

### Browser Support
* Mozilla Firefox 5+
* Google Chrome 14+
* Safari 5+
* Opera 11+
* Internet Explorer 8+
